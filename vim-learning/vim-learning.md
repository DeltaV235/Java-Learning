# VIM 笔记

#### 从normal进入insert

```shell
a   append
i   insert
o   open a line below
A   append after line
I   insert before line
O   append a line above
```

sp、vs                  可进行分屏编辑
% s/str1/str2/[g]       可进行文本[全局]替换；
normal模式下 v/V/^v进行可视化（光标/整行选中/块）操作
#### vim 快捷键 (insert mode)
ctrl + h            删除字符
ctrl + w            删除单词
ctrl + u            删除一行
ctrl + c / [        功能同Esc 切换insert和normal模式
